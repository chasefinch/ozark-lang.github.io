Open-Source Code

Ozark is maintained and developed on Github, including all code & documentation source, the sample apps, the full language specification, and the roadmap for future language development.

View Ozark on GitHub  

Community

Google Group

The Ozark Google Group is the official forum/mailing list. It is the primary place for announcements and discussion regarding the development and usage of the Ozark language.

IRC

Join #ozark on irc.freenode.net for informal conversation, help and commit notifications.

Founder

Ozark was founded in San Diego, California by Chase Finch, primarily in support of a venture to build visual software development tools. Find him on Twitter as @jchasefinch, Github as chasefinch, or by email at jchasefinch@gmail.com.
