Ozark is an elegant, open-source programming language. Through strict code standards, Ozark enables new tools for generating software.

The Ozark language is still in early development. The compiler is not yet available, and contributors are needed (especially those with compiler expertise.)

Ozark is open source. It is hosted and developed on GitHub.

EXAMPLE

Person Mountaineer

  Boots boots
  Hat hat
  Map map
  Location location

  prepare
    Boots boots initialize
    Hat hat initialize
    .boots < boots
    .hat < hat

  follow trail:Trail > success:Bit
    trail findNextWaypoint > waypoint:next
    .location trailTo waypoint:next > trail:newTrail
    . follow trail:newTrail > success:result
    success < result

  climb mountain:Mountain
    . prepare
    map findTrail > trail:Trail
    . follow trail:trail > success:result
    - if result
      . shout "Hello, world!"

Clean & Concise

Ozark code follows a strict methodology and narrow formatting rules, with no reserved words. This makes it easy to learn, and ensures that developers at all levels write standardized, readable code.

Authoritative

Ozark is both a programming language and a file storage format for applications that generate code. This makes it well suited to development with visual programming tools or by Artificial Intelligence.

Concurrent

Ozark automatically executes code in parallel, based on the inherent safety of the built-in language rules. As the availablility of multi-core processors increases, software written in Ozark will implicitly take advantage.

Smart

Ozark combines object-oriented, functional, dataflow, and procedural programming methods naturally, with innovative new approaches to take advantage of each type of computing.
